Conciliação de extratos bancários

Bem-vindo ao sistema de conciliação de extratos bancários utilizando arquivos OFX! Utilize as opções acima para navegar

Descrição de módulos

  • Home - Leva a página inicial do sistema
  • Nova Importação - Abre o módulo para fazer upload de arquivos OFX, processamento para remoção de duplicidade, e carga dos mesmos no banco de dados.
  • Consultar - Abre listagem com todas as operações salvas no banco de dados.
